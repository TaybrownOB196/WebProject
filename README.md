# WebProject

## Clone from githug
Powershell
* ps> cd {WORKING_DIRECTORY}
* ps> git clone https://github.com/TaybrownOB196/WebProject.git
* ps> cd WebProject

## Project Creation and Run
Powershell
* ps> dotnet new mvc -n ptoject.web -f netcoreapp3.1
* ps> cd ptoject.web
* ps> dotnet run

Navigate to https://localhost:5000
